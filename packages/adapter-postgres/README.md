# @otter/adapter-postgres

Postgres target adapter for `otter`. Implements the `Adapter` interface from `@otter/core` using
`Bun.sql` (no `pg` or `postgres.js`).

See [../../docs/adapter-postgres.md](../../docs/adapter-postgres.md) for full documentation.

## Install

Currently distributed via Bun workspace. See [../../docs/packages.md](../../docs/packages.md).
