# @otter/core

Shared interfaces, config loader, SQL recording context, DAG builder, compile pipeline, DAG
runner, node selectors, and state store. All other otter packages depend on this one; it has no
runtime deps outside Bun built-ins.

See [../../docs/configuration.md](../../docs/configuration.md) and
[../../docs/packages.md](../../docs/packages.md) for full documentation.

## Install

Currently distributed via Bun workspace. See [../../docs/packages.md](../../docs/packages.md).
