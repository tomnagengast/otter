# @otter/source-postgres

Postgres source for `otter`. Implements the `Source` interface from `@otter/core` using `Bun.sql`.
Extracts rows in paginated batches of 5 000.

See [../../docs/source-postgres.md](../../docs/source-postgres.md) for full documentation.

## Install

Currently distributed via Bun workspace. See [../../docs/packages.md](../../docs/packages.md).
